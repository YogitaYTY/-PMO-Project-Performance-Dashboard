PMO Project Performance Dashboard

Overview

This project demonstrates how Power BI can be used to monitor and improve Project Management Office (PMO) performance through interactive dashboards and KPI reporting. It provides a centralized view of project execution, budget utilization, schedule performance, resource allocation, and project risks, enabling data-driven decision-making.

⸻

Business Problem

Organizations often manage multiple projects simultaneously, making it difficult to monitor project health, identify risks, and ensure projects are delivered on time and within budget. Decision-makers require a single source of truth to track project performance and support effective governance.

⸻

Business Objectives

* Monitor overall project performance.
* Track budget versus actual costs.
* Measure schedule adherence.
* Analyze resource utilization.
* Identify high-risk projects.
* Support executive reporting with interactive dashboards.
